# Swagger

Swagger 是一个规范和完整的框架，用于生成、描述、调用和可视化 RESTful 风格的 Web 服务。

# 在项目中添加依赖

<!--swagger-->
<dependency>
    <groupId>io.springfox</groupId>
    <artifactId>springfox-swagger2</artifactId>
    <version>2.7.0</version>
</dependency>
<!--swagger ui-->
<dependency>
    <groupId>io.springfox</groupId>
    <artifactId>springfox-swagger-ui</artifactId>

# 创建Swagger配置类

/**
 * Swagger2配置类
 *
 * 通过@Configuration注解，让Spring来加载该类配置。
 * 再通过@EnableSwagger2注解来启用Swagger2。
 */
@Configuration
@EnableSwagger2
public class SwaggerConfig {
	 /**
     * 创建API应用
     * apiInfo() 增加API相关信息
     * 通过select()函数返回一个ApiSelectorBuilder实例,用来控制哪些接口暴露给Swagger来展现，
     */
    @Bean
    public Docket webApiConfig(){
        return new Docket(DocumentationType.SWAGGER_2)
                .groupName("webApi")
                .apiInfo(webApiInfo())
                .select()
                //过滤掉所有error或error.*页面
                .paths(Predicates.not(PathSelectors.regex("/error.*")))
                .build();

    }

    private ApiInfo webApiInfo(){
	/**
     * 创建该API的基本信息（这些基本信息会展现在文档页面中）
     * 访问地址：http://项目实际地址/swagger-ui.html
     */
        return new ApiInfoBuilder()
                .title("测试")
                .description("测试接口")
                .version("1.0")
                .contact(new Contact("java", "http://dczeroz.cn", "zddongc@outlook.com"))
                .build();
    }
}

# 结合SpringBoot进行测试

测试个人项目中的接口，此处不将逻辑代码写出，仅供接口已测试出结果

@RestController
@RequestMapping("/music/user")
public class MusicAdminController {

    @Autowired
    private MusicAdminService musicAdminService;

    @PostMapping("adminLogin")
    public R loginAdmin(@RequestBody MusicAdmin musicAdmin) {
        String token = musicAdminService.login(musicAdmin);
        return R.ok().data("token",token);
    }
}

# Postman

Postman 是一款功能超级强大的用于发送 HTTP 请求的 Chrome插件 。

collection在postman里面相当于一个文件夹，可以把同一个项目的请求放在一个Collection里方便管理和分享，Collection里面也可以再建文件夹。

# RestfulToolkit

RestfulToolkit是IDEA的一个插件

# 安装

在IDEA插件中，查找到RestfulToolkit并安装

此处推荐几款IDEA插件

Background Image Plus换IDEA背景，让你的IDEA变得闷骚闷骚的，人群中打代码最闷骚的那一个

Translation,省去打开翻译软件的时间，谁让英语不好，哭

使用也十分方便，直接点击想翻译的单词或句子，右键一键解决

温馨小提示：插件虽好，也不要贪多哦

# 使用RestfulToolkit

安装后，右侧会有RestServices侧边栏，点击后会显示当前项目所有请求地址，可以进行输入查询，然后会直接把请求方式，地址以及参数列出来，默认请求服务器为本机（localhost:8080）,可根据需求更改。
